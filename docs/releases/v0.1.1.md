# v0.1.1 Release Notes

## Highlights

- Redesigned the Unity Editor window so new users can follow a guided Start tab instead of guessing what each command does.
- Added separate Tables, Merge, Gate, and Output tabs.
- Added tooltips for important controls and copy-command buttons for reproducible CLI runs.
- Added project status rows for local config and registry files.
- Improved Unity-side CLI executable resolution for Windows PATH/PATHEXT setups.
