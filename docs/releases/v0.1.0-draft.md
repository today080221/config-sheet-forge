# v0.1.0 Release Notes

## Highlights

- Cross-platform `config-sheet-forge` CLI with `init`, `doctor`, `discover-root`, `new-table`, `sync`, `merge`, and `gate`.
- Shared C# core for semantic workbook models, portable validation, semantic hashing, schema review, and three-way merge.
- Feishu/Lark provider backed by `lark-cli`.
- Unity UPM package with an Editor window that references the shared core and runs CLI commands.
- Human-facing docs for configuration, safe editing, portable subset rules, merge policy, and CI gates.
- Direct Unity Package Manager install via Git URL with `?path=/packages/unity#v0.1.0`.

## Safety

- No real tenant links, owner routes, secrets, access tokens, or private table contents are included.
- Local provider configuration and cache files are ignored under `.config-sheet-forge/`.
- Root discovery recommends candidates only; it does not write or select roots.

## Known Gaps

- Run disposable Lark integration tests against non-private sample sheets.
- Add Unity edit-mode tests after the package is imported into a real Unity project.
