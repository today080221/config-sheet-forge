# v0.1.0 Release Notes

## Highlights

- 首版跨平台 `config-sheet-forge` CLI，包含 `init`、`doctor`、`discover-root`、`new-table`、`sync`、`merge`、`gate`。
- 共享 C# core，提供语义工作簿模型、便携 validation、语义 hash、schema review、三方合并。
- 基于 `lark-cli` 的飞书/Lark provider。
- Unity UPM 包，Editor window 引用共享 core 并运行 CLI 命令。
- 配置、安全编辑、便携子集、合并策略、CI gate 文档。
- 支持通过 Git URL `?path=/packages/unity#v0.1.0` 安装。

## 安全

- 不包含真实租户链接、owner route、secret、access token 或私有表内容。
- `.config-sheet-forge/` 下的本地 provider config 和 cache 已被 git 忽略。
- Root discovery 只推荐候选项，不写入也不选择 root。

## 已知缺口

- 需要一次性 Lark 集成测试。
- 需要 Unity edit-mode tests。
